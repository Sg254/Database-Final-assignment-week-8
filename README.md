##  Project Setup

### FastAPI
1. Clone repo
2. Create virtual environment & activate
3. Run `pip install -r requirements.txt`
4. Start server: `uvicorn main:app --reload`
5. Visit Swagger docs: `http://127.0.0.1:8000/docs`

## API Endpoints

### Patients (FastAPI)
- `POST /patients`
- `GET /patients`
- `PUT /patients/{id}`
- `DELETE /patients/{id}`
